# Instrucciones para Agentes

**LEE ESTO AL INICIAR CADA SESIÓN.**

1. **Lee** `/ROADMAP.md` completo — contiene el contexto, estado, y plan del proyecto.
2. **Identifica** la fase actual y el próximo paso.
3. **Al finalizar**, actualiza:
   - La tabla de sesiones en ROADMAP.md (fecha, logros, pendientes, decisiones)
   - El estado de funcionalidades que hayas modificado
   - El tema actual / próximo paso
4. **No trabajes en features sin actualizar el roadmap primero.**
5. **No borres** archivos de sesiones anteriores.
